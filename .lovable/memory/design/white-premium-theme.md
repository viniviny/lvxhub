Theme defaults to 'light'. Dark mode still available via toggle.

Color system is neutral monochrome:
- Background: pure white (#FFFFFF)
- Secondary/Muted: near-white gray (#F8F8F8 / #F5F5F5)
- Borders: light gray (#EBEBEB)
- Text primary: near-black (#111111)
- Text secondary: medium gray (#666666)
- Text muted: lighter gray (#666666)
- Accent = primary = near-black (#111111)

No blue accent colors in UI chrome. Info color (217 91% 60%) used only for data visualization.
Cards use white bg + subtle border, no heavy shadows.
Buttons: primary is black with white text, outline is transparent with border.
Sidebar: white bg, active item uses secondary bg, no colored left border effect (uses foreground).
